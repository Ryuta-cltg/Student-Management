＃受講生管理システム

本アプリケーションは、受講生情報の登録・一覧表示・更新することのできる簡易的な受講生管理システムです。
バックエンドはSpring Boot、フロントはThymeleafを使用しています。

---
## 📌 使用技術

- Java 17
- Spring Boot 3.x
- Spring MVC
- MyBatis
- Thymeleaf
- MySQL
- Gradle
- IntelliJ IDEA
- Git / GitHub

---

## 🎯 主な機能

- 受講生一覧表示（Thymeleaf + REST）
- 受講生新規登録（バリデーション付き）
- 受講生情報の更新機能
- コース情報の紐付け（受講生1名に対して複数コース可）
- エラーハンドリング（未登録のコース名に対して例外）
